# Sistema de Orçamento de Aluguel - Imobiliária R.M

Este é um projeto acadêmico de um sistema desenvolvido em Python 3 utilizando os conceitos de **Programação Orientada a Objetos (POO)**.

O objetivo do programa é calcular o orçamento mensal de aluguel para três tipos de imóveis (Apartamento, Casa e Estúdio), projetando o valor das parcelas do contrato imobiliário e gerando um arquivo `.csv` com a projeção detalhada ao longo de 12 meses.

## 🚀 Funcionalidades
* Cadastro e validação de dados de entrada.
* Geração automática de relatório de projeção em formato `.csv`.

## 🛠️ Tecnologias Utilizadas
* **Python 3** (Linguagem principal)
* **Programação Orientada a Objetos (POO)**
